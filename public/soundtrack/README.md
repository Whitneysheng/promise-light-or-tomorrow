# Soundtrack Stems

Place the rendered SuperCollider stems here:

- `01_wind_eflat_stem.wav`
- `02_d_natural_only_stem.wav`
- `03_bflat_bnatural_stem.wav`

The performer app plays these additively on the first three bar cues:

- bar 10: wind + E-flat
- bar 12: D natural only
- bar 18: B-flat, then B natural three seconds later

Render these as long enough files for the performance section, because the app
plays each stem once and leaves it running. Earlier stems are not stopped when
later cues are advanced.
